# Crowdloader
Crowloader is a tool to download translations files from crowdin.com.

## Installation
```bash
git clone https://github.com/ValDesign22/crowdloader.git
cd crowdloader
npm install

npm i -g .
```

## Usage
```bash
crowdloader --help

# Example
crowdloader --projectid <projectid> --apikey <apikey> (--project <folder>)
```
